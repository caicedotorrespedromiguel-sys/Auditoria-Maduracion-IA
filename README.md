# 🚀 Sistema de Maduración Tecnológica e IA

Este repositorio es una herramienta profesional para la auditoría de empresas. 
Basado en el **Diplomado Internacional en IA y Automatización**.

## 🛠 Cómo usar este sistema:
1. **Para realizar una auditoría:** Ve a la pestaña [Issues](../../issues/new/choose) y elige el formulario de Auditoría.
2. **Modelos Base:** Consulta la carpeta de archivos para ver los scripts HTML de reportes.
3. **Casos de Éxito:** Revisa los PDFs adjuntos (Calma, Casa Libertadoras, El Guane).

## 📊 Metodología
Evaluamos infraestructura, datos y talento humano para proponer Agentes IA y automatizaciones con Make/n8n.
